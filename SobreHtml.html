<HTML/>

HyperText Markup Language — a linguagem que dá estrutura e significado a toda página web.

O que é HTML?

HTML é a linguagem de marcação padrão para criar páginas na web. Ela descreve a estrutura de um documento por meio de tags — marcadores que dizem ao navegador o que cada parte do conteúdo representa.

Imagine um jornal impresso: ele tem título, subtítulo, parágrafos, imagens, listas. O HTML faz exatamente isso, mas para o ambiente digital, dizendo ao navegador: "isso é um título", "isso é um parágrafo", "isso é uma imagem".

💡

Você sabia? O HTML foi criado por Tim Berners-Lee em 1991. A versão atual, o HTML5, foi lançada em 2014 e trouxe tags semânticas, áudio, vídeo e muito mais.

Anatomia de uma Tag

Todo elemento HTML segue a mesma estrutura básica:

<p class="destaque"> Conteúdo aqui </p>
  • Tag (nome do elemento)
  • Atributo
  • Valor do atributo
  • Conteúdo textual

Tags Essenciais

Conheça as tags mais usadas no dia a dia:

<h1> … <h6>

Títulos hierárquicos. <h1> é o mais importante e deve aparecer uma vez por página.

<p>

Parágrafo de texto. O bloco mais comum para conteúdo textual em uma página.

<a>

Âncora/link. Com o atributo href, cria hiperlinks para outras páginas ou recursos.

<img>

Imagem. Usa src para o caminho e alt para descrição acessível.

<ul> / <ol>

Listas não-ordenadas e ordenadas. Cada item é envolto pela tag <li>.

<div> / <span>

Contêineres genéricos. <div> é bloco, <span> é inline.

Tags Semânticas

HTML5 introduziu tags com significado próprio, que deixam o código mais legível para humanos e máquinas:

<header>
<nav>
<main>
<article>
<section>
<aside>
<footer>
