Tecnologia #1

<HTML/>

HyperText Markup Language — a linguagem que dá estrutura e significado a toda página web.

Anatomia de uma Tag

Todo elemento HTML segue a mesma estrutura básica:

<p class="destaque"> Conteúdo aqui </p>
  • Tag (nome do elemento)
  • Atributo
  • Valor do atributo
  • Conteúdo textual

Tags Essenciais

Conheça as tags mais usadas no dia a dia:

<h1> … <h6>

Títulos hierárquicos. <h1> é o mais importante e deve aparecer uma vez por página.

<p>

Parágrafo de texto. O bloco mais comum para conteúdo textual em uma página.

<a>

Âncora/link. Com o atributo href, cria hiperlinks para outras páginas ou recursos.

<img>

Imagem. Usa src para o caminho e alt para descrição acessível.

<ul> / <ol>

Listas não-ordenadas e ordenadas. Cada item é envolto pela tag <li>.

<div> / <span>

Contêineres genéricos. <div> é bloco, <span> é inline.

Tags Semânticas

HTML5 introduziu tags com significado próprio, que deixam o código mais legível para humanos e máquinas:

<header>
<nav>
<main>
<article>
<section>
<aside>
<footer>

Por que semântica?

A semântica no HTML serve para dar significado aos elementos da página. Em vez de usar apenas tags genéricas, como div, utilizamos elementos que descrevem claramente o conteúdo, como header, main, section e footer.

Isso torna o código mais organizado, melhora a acessibilidade para leitores de tela e ajuda mecanismos de busca, como o Google, a entenderem melhor a estrutura e a importância de cada parte do site.

Você sabia? Sites que utilizam HTML semântico costumam ter melhor desempenho em SEO e oferecem uma experiência mais acessível para todos os usuários.
